Добавление одного файла в индекс(под версионный контроль)
```console
$ git add README
```

```console
$ git status
On branch master
Your branch is up-to-date with 'origin/master'.
Changes to be committed:
  (use "git restore --staged <file>..." to unstage)

    new file:   README
```

Добавление всех файлов в индекс 

```console
git add .
```
